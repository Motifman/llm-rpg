# 実験 01 — 背景画像生成プロンプト

`abandoned_hospital` のスポット背景を生成するためのプロンプト集。ランタイム上のスポットは `data/scenarios/abandoned_hospital.json` を正本とし、ここでは「引いた視点でも明らかにその場にあるもの」だけを入れる。

## 共通方針

- 人物は基本的に描かない。ただし `final_room` は拘束された澪の存在がスポット定義そのものなので、顔や表情を描き込みすぎないシルエット寄りで扱う。
- 血、怪物、過度な残酷表現は使わない。
- 文字は読めすぎない。読ませる情報はゲーム本文側で扱う。
- 画面は背景イラスト向けに、書き込みすぎず、探索可能オブジェクトの位置関係が分かる程度に留める。

## 共通スタイル指定

写実寄り、映画的、暗いが細部は読める、冷たい濃紺と灰色、差し色は非常灯の赤と古い照明の鈍い黄色、静かな心理ホラー、過度なゴア表現なし、背景イラスト向け、書き込みすぎない。

## 外観

静原総合病院の外観。地方都市の外れに建つ、かつては地域中核病院だった大きな総合病院の廃墟。夜、曇った月明かりの下、正面玄関は崩れた瓦礫と歪んだ自動ドアで完全に塞がれている。建物は昭和末期から平成初期の無機質なコンクリート造で、白かった外壁は灰色にくすみ、雨だれ、黒ずみ、ひび割れ、蔦、剥がれた案内板が目立つ。窓の多くは暗く、一部だけ割れており、内側から消えかけた非常灯の赤い光がにじんでいる。病院名の看板「静原総合病院」は半分だけ照明が死んでいて、文字の一部が読めない。前庭には放置された救急車用ロータリー、錆びた車止め、倒れたバリケード、濡れたアスファルト、水たまり、枯れた植え込みがある。全体の雰囲気は心理ホラーで、血や怪物は出さず、静けさ、記憶の欠落、何かが消された後の不自然な整然さを強調する。画面構図はやや低い視点から病院を見上げ、建物を大きく、空を狭く、左右非対称に配置する。

## スポット別プロンプト

### エントランスホール

静原総合病院の正面玄関ホール。崩れた瓦礫で自動ドアが完全に塞がれ、埃をかぶった受付カウンター、色褪せた院内案内板、壁に貼られた半ば破れた閉鎖告知の貼り紙が見える。貼り紙の文字は読めすぎない。かつて明るかったロビーが、今は月明かりと非常灯だけで沈んでいる。人物なし。

### 病棟受付

廃病院の病棟受付カウンター。散らばった診察券、古い案内書類、壁に残された退院記念写真の掲示板がある。写真は細部まで見せすぎず、ただ不自然に整った微笑みだけが印象に残る。静かで不穏な病院内装。人物なし。

### 1F ナースステーション

1F中央のナースステーション。カウンターには積まれたカルテ束があり、壁には当直表が貼られている。奥には一つだけ私物の残った職員デスクがあり、埃をかぶった写真立て、古いメモの束、少し開いた引き出しが見える。文字や写真の人物は判別できない程度に抑える。人物なし。

### 薬剤室

廃病院の薬剤室。金属製の薬品棚が並び、ほとんどの薬瓶は空か持ち去られている。棚板には剥がされたラベル跡が残っているが、文字は遠目には読めない。奥には扉が半開きの薬剤師ロッカーが見える。人物なし。

### リハビリ室

1Fのリハビリテーション室。大きな壁面鏡、平行棒、古いリハビリ器具が整然と残っている。鏡には黄ばんだテープが貼られ、壁の掲示板には患者用の名札がいくつか残っている。名札の文字は読めすぎない。人物なし。

### 2F ナースステーション

2Fスタッフ区画のナースステーション。申し送りノート棚、職員デスク、壁の当番表が残っている。机の上には古い申し送りノートが一冊置かれているが、本文は見えない。1Fより人の気配が薄く、職員だけが知っていた場所の静けさを出す。人物なし。

### 記録保管棚

2Fの記録保管室。面会記録、来院記録、紹介状控えのファイルが年代順に並び、一部の棚だけが焼け焦げている。棚の中には大きな面会記録簿と、クリップの付いた古い来院記録ファイルが見える。文字は読めない。人物なし。

### 職員休憩室

2Fの職員休憩室。開いたままの職員ロッカーが並び、多くは空になっている。古いテーブルの上には飲みかけのマグカップと、複数枚の書き置きが散らばっている。書き置きの文字は読めない。生活感はあるが、誰も戻ってこない静けさを中心に描く。人物なし。

### 書類庫

2Fの書類庫。並んだ棚のうち、一角だけが黒く焼け焦げ、隣の棚は手つかずで残っている。燃え跡は派手にせず、何かを選んで消した不自然さを見せる。煙の跡、紙片、暗い棚がある。人物なし。

### 院長室

2F奥の院長室。重厚な書斎机、本棚、大型金庫がある。病院内でここだけ古い邸宅の書斎のように整っていて、机の上に一冊の内部マニュアルが置かれている。上品だが息苦しい空間、暗い木目。人物なし。

### 3F 閉鎖病棟廊下

3F奥の閉鎖病棟へ続く中央廊下。扉は病室のように並んでいるが、廊下側の札は「301号室」ではなく、C-301、C-302、C-303のような記号で統一されている。用途表示や診療科名はなく、廊下の床には剥がれた扉札が一枚落ちている。最奥の扉には C-307 とだけある。文字は読めすぎない。無機質で冷たい隔離区、長い廊下。人物なし。

### 隔離区管理室

第三隔離区の管理室。壁には大きな区画図、机には古い制御パネル、周囲には管理記録のファイルが残っている。制御パネルの小さな赤いランプだけがまだ点いている。区画図の細かい文字や番号は読めすぎない。人物なし。

### 処置室

3Fの処置室。中央に処置台、横に金属の器具トレイ、古い処置記録簿が置かれている。医療器具は細かく描きすぎず、清潔だった部屋が長く放置された違和感を出す。冷たい白と灰色、静かな緊張感。人物なし。

### 最終処置室 C-307

3F最奥の最終処置室。中央の処置台に、拘束された若い女性の姿がある。ただし顔や表情は描き込みすぎず、シルエットに近く抑える。そばには古い医療ライト、心拍モニター、器具トレイがあり、壁際にはもう一台の空の処置台が見える。血や残酷描写は出さず、息が詰まるような静けさを表現する。

### 職員用裏口

職員が撤収時に使った病院裏手の鉄扉。狭いバックヤードに錆びた搬入口、古い段ボール、倒れた台車が残っている。扉の向こうから夜の外気と月明かりが少しだけ差し込んでいる。脱出直前の静かな緊張。人物なし。

### 外（脱出成功）

静原総合病院の裏手の空地。夜空の下、廃病院が月明かりに黒くそびえている。開いたままの裏口、濡れた地面、枯れた草、遠くに病院の暗い窓が見える。解決ではなく、ただ外に出たという余韻を重視する。静かな解放感と後味の悪さ。人物なし。
