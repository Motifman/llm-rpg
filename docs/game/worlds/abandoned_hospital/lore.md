## 1. 舞台

**静原（しずはら）総合病院**。地方都市の郊外にあった中規模の総合病院。外科・内科・精神科を備え、**重度トラウマ・解離症状・事故後の記憶障害に対する先進治療**で近隣では知られていた。ある年の秋、「医療スキャンダル」を理由に突如閉鎖。その後の詳細は行政記録からも欠落しており、建物は撤収作業が中途半端なまま放置され、現在は廃墟化が進んでいる。

- **第三隔離区（C 棟）**: 3F に位置する閉鎖病棟。公称は「重度解離患者の静穏病棟」だが、**実態は記憶再編処置の実施区画**。廊下側の扉札は病室番号のように見えるが、実際には処置工程の管理コードとして運用され、記録上の氏名は抑えられている。
- **撤収の不自然さ**: 職員の私物ロッカーはほぼ空、しかし患者の持ち物はそのまま残されている。書類の一部だけが完全に焼却され、他の記録は手付かず。つまり**何を消し、何を残しても構わないかを急遽選別した痕跡**が建物全体にある。

## 2. 真の実態

静原総合病院は、**特定記憶の切除・封鎖・再編を行う研究施設**を内部に抱えていた。
表向きは PTSD・解離・記憶障害の先進治療として患者を受け入れる。しかし実態として行われていたのは:

- 患者本人にとっても苦痛である記憶の**選択的除去**（これは治療として機能する側面もある）
- 同時に、**患者が持つ "施設にとって不都合な記憶"** の除去
- 処置の外部漏洩を防ぐための、**関係者・目撃者への予防的処置**

院内では処置対象の患者は「患者」でも「被験者」でもなく、書類上は **"対象者"** と呼ばれる。治療とは、**苦痛の除去であると同時に、真実の抹消**だった。

## 3. 中心人物

### 3.1 志木（しき）── 主人公 / 夜勤看護師

- 静原総合病院に勤めて数年。夜勤帯の担当看護師。
- 患者に寄り添うことを自分の支えにしてきた優しい人物。だが長く勤めるうちに、**説明のつかない処置**・**不自然な転院記録**・**番号で呼ばれる患者たち**に薄い違和感を抱いている。
- その違和感を深く追うことはしてこなかった。**見て見ぬふりをすることで、自分の仕事と生活を保ってきた**。
- 性別は明示しない（プレイヤー＝器の投影を妨げないため）。呼ばれる時は「志木さん」。

### 3.2 真島 澪（まじま みお）── 今夜搬送されてくる女性

- 学生時代からの友人。ここ数年は疎遠だった。
- 数年前、家族（姉・**真島 透子**）の治療の付き添いで、静原総合病院を一度だけ訪れたことがある。その際、**本来立ち入ってはいけない第三隔離区に迷い込み、処置中の何かを目撃した**。
- 直後に施設側に発見され、**記憶処置を受けた**。目撃した具体と、施設に来たこと自体も曖昧にされた。
- その後、日常生活を送っていたが、**最近になって処置の消し残しが再燃**している:
  - 知らないはずの**C-307 という記号**を夢で繰り返し見る
  - なぜか静原総合病院の風景が夢に出てくる
  - 強烈な既視感と、原因不明のフラッシュバック・自傷未遂・人物誤認
- 別の病院で対応困難と判断され、「重度解離・トラウマ記憶再燃対応」の専門施設として静原に搬送されてくる。**表向きは自然な転送、実態は本人の意思によって引き寄せられた帰還**。
- 志木は搬送カルテの名前を見て凍りつく。

### 3.3 真島 透子（まじま とうこ）── 澪の姉

- 数年前、静原総合病院で治療を受けていた。
- 具体的な治療内容は**黒塗りで明かされない**。診療科欄は「先進治療」とのみ記載。
- 透子のその後についても、本ワールドでは開示しない。澪の家族との関係も現在は疎遠（示唆のみ）。

### 3.4 渡良瀬（わたらせ）主任医師

- 第三隔離区の再編プロトコル責任者。
- 人物として直接登場はしない。痕跡（指示書・焼却残り・処方記録）でのみ示される。
- 今夜、志木に対して**澪の処置に立ち会うよう指示を出す**張本人。

### 3.5 久世（くぜ）院長

- 静原総合病院の経営責任者。
- 今夜、撤収と証拠完全削除の決定を下す。本人も今夜のうちに施設から離れ、以後所在不明。

### 3.6 同僚看護師（今夜の去就）

- **樋口**: 撤収作業（書類焼却・薬品撤去）に加担する。志木と目を合わせない。
- **苅田**: 今夜の途中で姿を消す。制服とロッカーの鍵だけが残されている。
- **園田**: 志木の先輩。夜中、志木にすれ違いざま一言だけ残して去る──「**見なかったことにして**」。

## 5. 中心となる謎

- **主人公（志木）はこの夜の後どうなったのか** — 現代の廃墟に残る "誰かの痕跡" が暗示するが確定しない
- **第三隔離区の残存患者は救われたのか** — 今夜の "次段階" 処置がどこまで進んだかは不明
- **澪が以前目撃した具体的内容** — 断片（消毒液の匂い、光、拘束帯の軋む音、声にならない悲鳴）だけ。"人間に行ってはいけない何か" という倫理的確信だけが残る
- **澪の姉・真島 透子の治療内容** — 記録は黒塗り。本ワールドでは開示しない
- **施設の最初期の実態** — いつから記憶再編が行われ始めたのか、久世院長と渡良瀬医師のどちらが主導したのか、記録が意図的に欠落している

## 6. テーマと演出ルール

- **テーマ**: 救えなかった罪悪感 → 「忘れていい」と自分に許可を出す。
  - ただし本ワールドでは**変奏**: 「救うとは何か」「忘れさせることは救いか」という問いが前面に立つ。
  - 主人公の最終選択は、どれを選んでも**清くない**。手術 = 多数の継続的犠牲／逃走 = 友人の苦痛継続と不確実な告発／自己消去 = 全員残される。
- **ホラー演出の節度**: ジャンプスケアも超自然要素も使わない。**人為の痕跡・記述の不自然さ・人間の選択**で不穏さを作る。
- **メタ干渉**: Stage 1 のため、システムメッセージは普段どおり。少女のフラッシュバック演出で**1 回だけ**人称をぶらす（[gate_girl_link.md](./gate_girl_link.md) §4）。
- **追体験の記述技法**: 現代パートで特定の残留物に触れた瞬間、人称が「わたし（志木）」に切り替わる。**超自然ではなく語りの技法**として扱う。現代に戻るときは、当時触れていた物が現代の廃墟にも残っている、という形で視点橋を作る。
