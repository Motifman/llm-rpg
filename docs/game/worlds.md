# 脱出ワールド設計（実験ワールド）

本書は **「実験」= 脱出ワールド** の草案集と、ワールド全体に通底するメタ構造を整理したもの。
世界観の正本は [story_concept.md](./story_concept.md) §2、画面遷移の正本は [virtual_world_game_design_plan.md](./virtual_world_game_design_plan.md) §3.2。
キャラクター単票は [characters/](./characters/) 配下。

> プレイヤー向け UI ラベルは「**実験**」系で統一する（[story_concept.md](./story_concept.md) §1.4 / [README.md](./README.md)）。

---

## 1. ワールド共通の構造

各ワールドは次の 4 要素で構成する。

| 要素 | 役割 |
|------|------|
| **舞台 (Stage)** | 一見現実に似て、夢のように捻じれた風景（駅・宴会場・書庫・灯台・温室 等）|
| **対象住人 (Subject)** | このワールドの**記憶の持ち主**。荘園の住人の中の一人。|
| **隠れたテーマ (Theme)** | 住人の未練・トラウマ・欲求の核。プレイ中は明示しない。|
| **エピローグ変化 (Shift)** | ラン終了後の住人の**心境/態度の大きな変化**。エピローグで必ず描く。|

**フロー**: ワールド選択 → キャラ選択 → 導入ノベル → 脱出シミュレーション → **エピローグ** → リザルト → 荘園ハブ
（[virtual_world_game_design_plan.md](./virtual_world_game_design_plan.md) §3.2）

---

## 2. メタテーマ —— プレイヤー＝器（Vessel）

**裏のテーマ**: ラン毎にプレイヤーは住人の記憶・トラウマを少しずつ「**器**」として受け取り、変質していく。
ゲームの中盤以降、プレイヤー側にも変化の兆候が表面化する。

| ステージ | 顕在度 | 演出例 |
|---------|--------|--------|
| **Stage 1: Subtle**（実験 01–02） | 0 | 少女の語りに「あなた“が”覚えている気がする」と一瞬の言い違い |
| **Stage 2: Emerging**（実験 03–04） | 中 | システムテレメトリに `SUBJECT: PLAYER?` のグリッチ。エピローグ後、プレイヤー独白が一行差し込まれる |
| **Stage 3: Exposed**（実験 05 / 終盤） | 高 | プレイヤーが「自分が誰だったか思い出せない」感覚に到達。少女が初めて**プレイヤーを名指しで呼ぶ** |

このメタは**システムメッセージの語彙のずれ**（[story_concept.md](./story_concept.md) §4.3）と一体で運用する。

## 3. ワールド草案（5本）

| # | ID | 邦題 | 対象住人 | テーマ | エピローグ変化 |
|---|----|------|---------|--------|--------------|
| 01 | `abandoned_hospital` | 廃病院からの脱出 ―― 白鷺病院の記憶 | 門前の少女 | **救えなかった罪悪感** | 「忘れていい」と初めて自分に許可を出す |
| 02 | `snowbound_station` | 雪原の駅 ―― 終電の残響 | （未割当・第二解放枠） | **待ち続けた未練** | 「もう来ない」と認め、ベルの音から解放される |
| 03 | `endless_banquet` | 終わらない宴会場 | （未割当） | **承認欲求 / 終わらない期待** | 他者の評価から距離を置き、自分の食卓を持つ |
| 04 | `underwater_library` | 水面下の書庫 | （未割当） | **忘却と執着** | 守るべき1冊を選び、残りを手放す |
| 05 | `mirrored_greenhouse` | 鏡の温室 | （未割当） | **理想化された自己像** | 鏡像を割り、欠けた自分を受け入れる |

> 02–05 の対象住人は脱出クリアによる視点解放（[story_concept.md](./story_concept.md) §1.4）に合わせて確定する。
> 02 以降の「現実離れした夢の場」案は [story_concept.md](./story_concept.md) §2.2 を起点にしている。

### 3.1 実験 01: 廃病院からの脱出 ―― 白鷺病院の記憶

- **舞台**: 朽ちた病院。非常灯が一定間隔で点滅し、廊下の奥に出口の標識がかすかに見える。住人たちは協力して脱出を試みる。
- **対象**: 門前の少女（[gate_girl.md](./characters/gate_girl.md)）
- **テーマ**: 救えなかった他者への罪悪感。出口は見えているのに「振り返れない」設計。
- **ゲームプレイの核**: **協力 / 推理 / ホラー**。複数住人 NPC との連携選択、患者室に残された「自分宛の手紙」を読むか捨てるか。
- **エピローグ変化**: 少女が「忘れていい」と自分に初めて許可を出す。BGM が一段静かになる。
- **少女の誘導台詞（選択画面）**:
  - 「次に潜るのは……ここ。あなたが選んで。」
  - 「大丈夫。わたしも一緒に視ている。」

### 3.2 実験 02: 雪原の駅 ―― 終電の残響

- **舞台**: 雪に半ば埋もれた無人駅。時計は止まり、行き先表示は読めない言語。列車の音だけが遠くで鳴る。
- **テーマ**: 待ち続けた未練。誰を待っていたのかを思い出せた時のみ脱出口が開く。
- **ゲームプレイの核**: **時間 / 探索 / 選択**。ベンチに座って待つ／ホームを歩く／別の乗客に話しかける。
- **エピローグ変化**: 「もう来ない」と認める。荘園に戻った後もベルの音に一瞬だけ反応する癖が残る（[story_concept.md](./story_concept.md) §2.2 例1）。

### 3.3 実験 03: 終わらない宴会場

- **舞台**: 永遠に続く晩餐。客は減らず、料理も減らない。誰が主催者かは分からない。
- **テーマ**: 承認欲求。他人に好かれようとするほど出口が遠ざかる。
- **ゲームプレイの核**: **会話 / 役割 / 距離感**。給仕に立つか客として座るか、誰の隣に座るか。
- **エピローグ変化**: 評価から距離を取り、自分のための食卓を持つ（[story_concept.md](./story_concept.md) §2.2 例2）。
- **メタヒント**: ここから**プレイヤー側に器の自覚**が芽生え始める（テレメトリの `SUBJECT` が一瞬 `PLAYER?` にグリッチ）。

### 3.4 実験 04: 水面下の書庫

- **舞台**: 螺旋階段を下った先の書庫。水位が少しずつ上昇し、棚の本が沈んでいく。
- **テーマ**: 忘却と執着。すべてを救おうとすれば沈み、自分の本を捨てれば心が傷つく。
- **ゲームプレイの核**: **取捨選択 / インベントリ**。1ターンに1冊しか高所へ運べない。
- **エピローグ変化**: 守る本を1冊だけ選ぶ。残りを手放す痛みを引き受ける（[story_concept.md](./story_concept.md) §2.2 例3）。

### 3.5 実験 05: 鏡の温室

- **舞台**: 鏡張りの温室。鏡の中の自分だけが少し違う表情をしている。
- **テーマ**: 理想化された自己像。鏡像と話すほど自分が薄くなる。
- **ゲームプレイの核**: **対話 / 自己選択**。鏡像の言葉に同意するか反論するか。
- **エピローグ変化**: 鏡を割る。割れた破片の中に「今の自分」を初めて見つける。
- **メタヒント**: ここで**プレイヤー＝器**が**ほぼ露見**する。少女が初めてプレイヤーを名指しで呼ぶ。

## 4. ワールド数とエンディングの想定

- 初期実装スコープは **05 ワールド**。
- **05 のエピローグ後** に、グローバルなエピソードシークエンス（プレイヤー視点の独白／創造者の最後の記録）を挿入する余地を残す。
- 分岐エンディングは MVP では持たず、**「最後にどの住人視点を選ぶか」** による短いエピローグ差分で代替する。
