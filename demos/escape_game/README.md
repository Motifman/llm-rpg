# 廃病院からの脱出 ―― 白鷺病院の記憶

ホラー脱出ゲームのデモ。スポットグラフワールド（Step 1〜7）の全機能を統合し、
シナリオ JSON から読み込んだデータで観測→行動のゲームループを実行する。

## 実行方法

```bash
source venv/bin/activate
python -m demos.escape_game.run_escape_game
```

## 概要

### シナリオ

かつて地域医療の要だった白鷺総合病院。1987年に院長・鶴見博士の極秘延命研究が発覚し閉鎖。
数十年後、廃墟探索中に閉じ込められたプレイヤーたちが脱出を目指す。

- **スポット数**: 8（エントランス、廊下、手術室、院長室、地下室、隠し通路、非常口、外）
- **アイテム数**: 10（鍵アイテム2、ツール2、ドキュメント2、ロアアイテム4）
- **プレイヤー数**: 2（探索者A/ユウキ、探索者B/マコト）
- **勝利条件**: 全員が外に出る
- **敗北条件**: 120ティック以内に脱出できない

### パズルフロー

```
受付の引き出し → [院長室の鍵]
                       ↓
ナースデスク → [手術記録簿]
                       ↓
院長室 → 机 → [院長の日記] + read_diary フラグ
         ↓
         金庫（日記+記録→暗証番号1987）→ [非常口の鍵]
                                              ↓
器具棚 → [錆びたメス]                        ↓
              ↓                               ↓
地下室の壁の亀裂 → メスで広げる → 隠し通路    ↓
                                   ↓          ↓
                              非常口の扉 ← 鍵で開錠
                                   ↓
                                  外 → 🎉 脱出成功
```

※ 別ルート: 腐食剤（手術室の薬品キャビネット）で非常口の錠前を溶かすことも可能。

### デモが表示する内容

各ティックで以下を表示する：

1. **\[SYSTEM PROMPT\]** — LLM エージェントが受け取るシステムプロンプト
2. **\[OBSERVATION\]** — 現在地の説明、雰囲気、見えるオブジェクト、接続先、所持アイテム、フラグ
3. **\[AVAILABLE TOOLS\]** — LLM が選択可能なツール一覧（`spot_graph_travel_to`, `spot_graph_explore`, `spot_graph_interact`, `speak`, `shout`）
4. **\[ACTION RESULT\]** — アクション実行結果のメッセージ

## シナリオデータ

`data/scenarios/abandoned_hospital.json` にシナリオ定義 JSON がある。
このフォーマット（`scenario_format_version: "1.0"`）は以下で構成される：

| セクション | 内容 |
|---|---|
| `metadata` | シナリオのメタ情報（タイトル、テーマ、難易度等） |
| `item_specs` | アイテム定義（文字列ID、名前、説明、カテゴリ） |
| `spots` | スポット定義（位置、説明、雰囲気、interior含む） |
| `connections` | スポット間接続（通行条件、双方向性、音透過率） |
| `players` | プレイヤー初期配置 |
| `game_end_conditions` | 勝利・敗北条件 |
| `initial_flags` | 初期ワールドフラグ |

## アーキテクチャ

```
data/scenarios/abandoned_hospital.json
         ↓
ScenarioLoader (infrastructure/scenario/)
         ↓ ScenarioLoadResult
EscapeGameRuntime (demos/escape_game/)
         ↓ InMemory repositories を構築
    ┌────┴────────────────────┐
    │ Application Services    │
    │  - Movement             │
    │  - Interaction          │
    │  - Exploration          │
    │  - CurrentStateBuilder  │
    └────┬────────────────────┘
         ↓
    Domain Layer (world_graph/)
```

## ストーリー要素（ネタバレ注意）

シナリオには脱出に直接関係しないロアアイテムや演出が含まれている：

- **患者の腕輪** — 名前が塗りつぶされた入院患者の識別バンド
- **色褪せた写真** — 開院記念の集合写真。カメラを見ていない男がいる
- **割れた注射器** — 異様な甘い匂いのする注射器の残骸
- **看護師のメモ** — 「地下には絶対に行くな」と書かれた警告
- **壊れた監視モニター** — 地下カメラの記録テープ紛失の報告メモ
- **手術台の爪痕** — 被験体No.7が刻んだアルファベット
- **配電盤** — 数十年間稼働し続ける「生命維持装置」のブレーカー
- **冷却装置CR-7号** — 被験体No.7と同じ番号のクライオ保存ユニット
- **チョークの矢印** — 看護師・佐伯が残した脱出の手がかり

これらの断片から、鶴見博士の延命研究と被験体No.7の「成功」が暗示される。
