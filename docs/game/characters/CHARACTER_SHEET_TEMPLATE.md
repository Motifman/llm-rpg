# キャラクター設定シート（テンプレート）

物語上の役割と実装・演出でブレないよう、**世界内でキャラが知っていること**と**作者メモ**を分けて書く。

---

## 基本情報

| 項目 | 内容 |
|------|------|
| **仮コード名** | 実装・ファイル名用（例: `gate_girl`） |
| **表記名** | 画面・シナリオでの呼び名（未確定なら「未定」） |
| **初登場** | プロローグ / 章番号 / イベント名 など |
| **所属** | 荘園住人 / 外部 / 不明 など |

---

## 世界内の自己認識（キャラの主観）

- 自分は何者だと思っているか（人間、屋敷の一員など）
- 荘園・記憶喪失・閉じ込めについて何を知り、何を知らないか
- 仮想世界・実験・「インスタンス」などの概念を持ちうるか（通常は持たない）

---

## 作者メモ（プレイヤー・制作側のみ）

- 創造者（博士）の分類での位置づけ（例: 未練から抽出された人格の具現、実験体）
- 真実の正体（未確定なら仮説を列挙）
- 他キャラ・プレイヤー・システムとの関係で伏せたいこと／早めに見せたいこと

---

## 目的と動機

- **表向きの目的**（キャラが口にする・行動で示す動機）
- **裏の目的**（自己理解がある場合のみ。なければ「なし」）

---

## 物語上の関数

- このキャラが担う**シーン機能**（例: 初回接触の顔、疑念の種を蒔く、コミックリリーフ）
- **変化の余地**（物語後半で揺らぐ可能性）

---

## 口調・振る舞い

- 一人称・二人称の傾向、語彙、避ける言い回し
- ストレス時の変化

---

## 未決事項・検討メモ

- 決め切れていない選択肢（A/B 案）
- 依存する他設定（誰が外とコンタクトしたか、など）
