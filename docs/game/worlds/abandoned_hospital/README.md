# 実験 01: 廃病院からの脱出 ── 消し残された夜

実験 01「`abandoned_hospital`」のシナリオ管理ハブ。世界観・歴史・スポット・フラグ・少女との接点を Markdown 側で**一元的にデザインし**、ランタイム JSON へ片方向で反映する運用にする。

> **本シナリオは 2026-04 に全面刷新された**。旧設定（白鷺総合病院・鶴見博士・被験体 No.7・看護師佐伯・冷却装置 CR-7・1987 年事件）は全て破棄されている。本ファイル以下が新設定の正本。

## 1. シナリオの骨子

- **公称**: **静原（しずはら）総合病院** ── 重度トラウマ・解離症状・記憶障害の先進治療で知られる地域中核病院
- **実態**: 患者の特定記憶を切除・封鎖・再編する研究施設。治療は苦痛除去と真実抹消を兼ねる
- **追体験する視点**: 夜勤看護師 **志木（しき）**。見て見ぬふりで勤めてきた共犯者
- **"今夜"**: **真島 澪（まじま みお）** が、以前受けた処置の消し残しに導かれて自発来訪した夜。施設は証拠完全削除と撤収を決断する
- **廃墟化**: この一夜の撤収が廃墟化の引き金。現代パートの廃墟は "あの夜の結果"
- **メタ顕在度**: Stage 1 (Subtle) 厳守。超自然要素は使わない

### 改変ワークフロー

1. **md 側を先に書き換える**（lore / map / flags_items / gate_girl_link）。
2. 影響を受ける `data/scenarios/abandoned_hospital.json` を更新する。
3. ランタイム側が壊れていないか `pytest tests/e2e/test_escape_game_puzzle_flow.py` で確認する。
4. ID を変える場合は md → JSON の順で grep し、参照を全更新する。

> 後方互換のための旧 ID 残しは**しない**。改名したら旧 ID は削る（[implementation-discipline.mdc](../../../../.cursor/rules/implementation-discipline.mdc) 参照）。

### あの夜のこと（プレイヤー＝器の記録より）

崩落音が止んだとき、僕は**静原総合病院のエントランスホール**に立っていた。正面入口は瓦礫で塞がっている。戻る道はない。

この病院が「医療スキャンダル」で閉鎖されたことは知っていた。だが詳細は行政記録からも欠落しており、建物はただ放置されていた。今、こうして中に入ってみると──**撤収の不自然さ**が最初に目に付く。職員のロッカーは空。しかし患者の私物は部屋に置かれたままだ。書類の一部だけが完全に焼却され、他の記録は手付かず。**何を消し、何を残していいかを急遽選別した痕跡**だった。

1F の病棟受付には、患者たちの掲示写真が並んでいた。**複数人の側頭部に、同じ位置の縫合痕**があった。最初は偶然だと思った。ナースステーションでカルテ束を見つけたとき、主訴欄が**全員同じ筆跡で「不明」**と書かれているのに気付いた。奇妙だ、程度には思った。リハビリ室の鏡に「**あなたの名前は──**」で終わる古いテープが貼られていた。空欄だった。

1F のナースステーションに座って当直表を眺めていたとき、視界がにじんだ。気が付くと──**わたし**は、この同じ机の前に座っていた。夜勤入りの準備をしていた。名札を見る。**志木**。当直表が通常と異なる。今夜は夜勤がわたし一人に集中している。

廊下の向こうから、緊急搬送の報せが届いた。搬送カルテの名前を見たわたしは、凍りついた。**真島 澪**。見覚えのある名前だった。

わたしに戻り、机の引き出しを開けた。古い日記があった。ところどころ、**ページが白く抜けている**。抜けた日付は、わたし自身にも思い出せない。埃だらけの写真立てに、澪と二人で笑う自分（志木）が写っていた。裏に「またね」の走り書き。

2F に上がった。申し送りノートには「再燃前に処置完了」「関係者との接触に注意」。薬剤室の棚には**「R-7（想起誘導剤）」「パーサ-9（人格安定補助剤）」「リコール-β（記憶再編剤）」**。この病院は、記憶を扱う研究をしていた。L1 で見た違和感の全てが、ここで**意味を書き換えられる**。縫合痕の一致、主訴「不明」、鏡のテープ──全員が記憶処置を受けていた患者だった。

記録保管棚で、澪の姉・**真島 透子**の来院記録を見つけた。診療科欄は「先進治療」とだけ。主治医欄は黒塗り。澪は数年前、この姉の付き添いで病院を訪れていた。そしてその時、**本来立ち入ってはいけない第三隔離区に迷い込んだ**。直後に発見され、処置を受けた。それが、最近になって消し残しが再燃している。

院長室で『再編プロトコル 第三版』を見つけた。金庫のダイヤルは、症例番号 **0307**。少女が、何度もその番号に反応していた。**C-307**。

3F の第三隔離区では、病室番号のように見えた扉札が、区画図と照合すると処置工程の管理コードだった。処置室に入ると、わたしはまた**志木**に戻った。過去の処置記録と、自分自身が押した "処置完了" の判子の控えがそこにあった。**わたしは見て見ぬふりをしてきたどころではない。実際に加担してきた**のだ。

職員休憩室で同僚たちの今夜の去就を知った。樋口は撤収作業に加担していた。苅田は姿を消した。園田はすれ違いざま、一言だけ残した──「**見なかったことにして**」。院長室の鍵を、わたしに押し付けながら。

---

わたし（志木）は、最終処置室の扉の前に立っていた。中には、拘束された澪がいる。処置灯だけが白く点いている。

**手術を行えば**、澪はこの施設もわたしのことも完全に忘れて、日常に復帰する。施設は守られ、**他の多くの患者が今後も処置され続ける**。そしてわたしは、澪に忘れられる側になる。
**手術をせずに逃がせば**、澪の障害は残るが、徐々に思い出している真相の記憶は保持される。告発が成立する可能性がある。**多くの人が救われるかもしれない**。ただし、施設を裏切ったわたし自身がこの後どうなるかは分からない。
**あるいは、わたし自身が記憶処置を受ける**こともできる。今夜のことは全て無かったことになる。わたしは翌朝からまた、この病院で働き続ける。澪は予定通り処置され、日常に戻る。二人の関係も消える。

---

どれを選んでも、清くなかった。

僕（プレイヤー＝器）は、志木の選択を**ただ見届けた**気がする。あるいは、**その選択を自分が下した**気がする。どちらだったのか、今でも分からない。

裏口から外に出た。振り返ると、静原総合病院は月明かりに黒くそびえていた。──はずだった。ただ、振り返った一瞬だけ、建物の姿は選んだ選択によって少しだけ違って見えた気がする。**あの 1F ナースステーションだけ、異様に整頓されている**ようにも、**裏口が開いたまま風に煽られている**ようにも、**一瞬だけ稼働中の病院の姿**にも、見えた。

僕たちは何も解決していない。**主人公（志木）がその後どうなったのか**、**第三隔離区の患者たちが救われたのか**、**澪が以前目撃したものが何だったのか**、**透子の治療が何だったのか**──何ひとつ確かめていない。**ただ、出てきただけだ**。

それでも、少女はどこか、**何かを許せたような顔**をしていた。

## 5. 現スコープでの確定事項

- 対象住人: **門前の少女**
- 隠れたテーマ: **救えなかった罪悪感**（本ワールドでは「救うとは何か／忘れさせることは救いか」へ変奏）
- エピローグ: **「忘れていい」** を主人公の選択に応じて 3 種のトーンで分岐
- メタ顕在度: **Stage 1 (Subtle)** — 超自然要素は使わない。人称ぶれは追体験 4（処置室）で 1 回のみ
- ワールド ID: `abandoned_hospital`（既存踏襲、ランタイム影響最小化）
